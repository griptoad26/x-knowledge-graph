### Added
- Semantic search functionality
- Bidirectional linking
- Analytics dashboard
- PKM exports (Notion, Evernote, Markdown)
- AI export parsers (ChatGPT, Claude, Gemini)
- Brave search history parser
- Production log parser (prodlog format)

### Changed
- Improved data path resolution
- Unicode encoding fixes for Windows
- Grok parser now recurses subdirectories

### Fixed
- Desktop data path resolution
- Grok parser subdirectory recursion
